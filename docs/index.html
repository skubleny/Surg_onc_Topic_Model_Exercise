Methods note

Topic assignments, labels, embeddings, temporal summaries, and aggregate statistics are precomputed before deployment. The website itself is static, meaning the browser handles figure interactivity without requiring a live Python or R server.

Suggested workflow

  1. Run BERTopic in Python.
  2. Export interactive figures as standalone HTML files.
  3. Place Plotly HTML files directly in the figures/ folder.
  4. Place the datamap HTML and its required datamap_*.zip files together inside figures/topic_landscape/.
  5. Push the folder to a GitHub repository.
  6. Enable GitHub Pages for the repository.
